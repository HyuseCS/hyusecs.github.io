# Product

## Register

brand

## Users
Recruiters and hiring managers looking for professional proof and technical competence, and the developer community looking for technical depth, craft, and inspiration.

## Product Purpose
A high-craft personal portfolio that showcases Christian John Legaspi's capabilities as a Full Stack Developer and CS student. It serves as both a resume-plus and a technical demonstration of system telemetry and frontend precision.

## Brand Personality
- **Terminal-First**: Explicitly developer-centric, utilizing monospaced fonts and command-line aesthetics.
- **High-Craft**: Obsessive attention to detail in animations, telemetry data, and system-level status.
- **Robust**: Real-time integration (GitHub API) that proves technical skill through live data rather than static claims.

## Anti-references
- **Generic AI Portfolios**: Avoid the "AI slop" aesthetic—no soft pastel gradients, no generic floating cards, no rounded "SaaS-lite" templates.
- **Corporate Templates**: No generic Bootstrap or "professional blue" agency looks.

## Design Principles
- **Show, Don't Tell**: Use live telemetry and working code patterns (typewriters, terminal logs) to prove expertise.
- **Developer-Centric**: Optimize for the mental model of an engineer—precise, information-dense but legible.
- **The Machine Aesthetic**: Embrace the raw materials of computing—grids, terminal prompts, and status daemons.

## Accessibility & Inclusion
Targeting WCAG AA compliance. Focus on robust keyboard navigation (already initiated) and respecting user preferences for motion and contrast.
